'com_s.py' is the main script.
Create a telegram bot using 'botfather' and add your bot token and chat ID in the script.
Run it in MintOS or Ubuntu(Zorg) as they have low hardening.
Use any command with 'sudo' in a different terminal for the script to get access and create a new user with all the permissions.
After running the commands the desired output may take some time(1-2 minutes).
The script will send status messages to the telegram bot once it is ready and installed and everytime the user turns on the system.
Control everything through the bot with the commands in the menu.
